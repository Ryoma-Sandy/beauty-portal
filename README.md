# 美容医療 学会・勉強会ポータル

美容皮膚科・美容外科に関する最新の学術集会、ハンズオンセミナー、勉強会情報を発信するポータルサイトのフロントエンド実装です。

## プロジェクト概要
美容医療従事者が、自身の専門分野や活動地域に合わせて、必要な学びの場（学会、オンライン講座、ハンズオンセミナーなど）を効率よく検索・発見できるWebページです。

## 主な機能
- **イベント一覧の動的表示**: JavaScriptを使用して、イベント情報をカード形式でグリッド配置。
- **絞り込み・検索機能**:
  - **キーワード検索**: 学会名やテーマなどのテキスト検索
  - **カテゴリフィルター**: 美容皮膚科、美容外科、総合・その他
  - **開催地域フィルター**: 関東、関西、オンラインなど
- **レスポンシブデザイン**: スマートフォンやタブレットからでも快適に閲覧できるレイアウト。
- **プレミアムなUI/UX**: ゴールドやパールカラーを基調とし、各種アニメーション（ホバー時の浮き上がり、順次フェードインなど）を取り入れた洗練されたデザイン。

## 使用技術
- **HTML5**: セマンティックなマークアップと構造定義
- **CSS3**: Vanilla CSSでのスタイリング（Flexbox/Gridレイアウト、カスタムプロパティを用いたテーマ管理、CSSアニメーション）
- **Vanilla JavaScript**: モックデータの定義、DOM操作、リアルタイムの非同期フィルタリング処理
- **外部リソース**:
  - Google Fonts (`Noto Sans JP`)
  - FontAwesome (アイコン)
  - Unsplash (仮のヒーロー画像)

## ファイル構成
- `index.html`: メインページの構造。ヒーローセクション、検索UI、イベント表示エリアを定義。
- `style.css`: 配列、配色、アニメーションなど、全体のスタイリング要素を集約。
- `main.js`: `eventsData`（モックデータ）の定義と、画面ロード時の表示処理、および検索ボックス・セレクトボックスの変更を検知してカードを出し分けるロジック。
